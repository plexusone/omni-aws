## [v0.7.0] - 2026-04-25

### Highlights

- Consolidated to single Go module at repository root
- Unified AWS provider package with both LLM (Bedrock) and Storage (S3) adapters

### Breaking

- Module path changed from `github.com/plexusone/omni-aws/omnillm` to `github.com/plexusone/omni-aws`
- Module path changed from `github.com/plexusone/omni-aws/omnistorage` to `github.com/plexusone/omni-aws`

### Upgrade Guide

- Import paths remain the same: `github.com/plexusone/omni-aws/omnillm` and `github.com/plexusone/omni-aws/omnistorage`
- Run `go get github.com/plexusone/omni-aws@v0.7.0` to update dependency

### Changed

- Removed nested `omnillm/go.mod` and `omnistorage/go.mod` in favor of root-level `go.mod`
- Consolidated changelogs from per-module to single repository changelog
- Simplified CI workflows (removed multi-module configuration)

### Internal

- Previous releases: `omnillm/v0.6.0` (Bedrock adapter), `omnistorage/v0.1.0` (S3 adapter)
